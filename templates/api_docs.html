Методы api

  • id

    • Возвращает id всех звуков. Также есть параметр soundname, который принимает название звука. В таком случае возвращает айди всех звуков с этим названием.

  • sound/int:sound_id

    • Возвращает звук в виде байтовой строки, закодированной в base64.
